FAQ

O que é HTML?

HTML é a linguagem usada para criar e estruturar páginas web.

O que é CSS?

CSS estiliza o HTML e define cores, fontes e layouts.

O que é JavaScript?

JavaScript adiciona interatividade às páginas web.

O que é semântica em HTML?

É o uso correto de tags para indicar a função de cada conteúdo.

Por que usar <aside>?

Para conteúdo complementar, como barra lateral de informações.
